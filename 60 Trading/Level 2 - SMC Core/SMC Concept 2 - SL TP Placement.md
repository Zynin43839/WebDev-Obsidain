# SMC Concept 2 — SL (Stop Loss) & TP (Take Profit)

---

## 🔹 SL = Stop Loss

### ความหมาย
SL คือ **เส้นแบ่งระหว่าง "เดาถูก" กับ "เดาผิด"** — เมื่อราคามาถึงจุดนี้ ออเดอร์จะปิดอัตโนมัติเพื่อจำกัดขาดทุน

### จุดประสงค์
1. **ป้องกันความหวัง** — ไม่ให้ถือต่อเพราะ "เดี๋ยวมันกลับ"
2. **แยกเทรดเดอร์กับนักพนัน** — ขาดทุนจำกัด = เล่นต่อได้
3. **ไม่ต้องคิดตอนเครียด** — ตั้งไว้ตอนใจเย็น

### วิธีใช้ — 3 แบบ

#### แบบ A: SL = ใต้ OB 1-2 pip
| หัวข้อ | รายละเอียด |
|--------|-----------|
| ใช้ตอน | OB แรง + confluence เยอะ |
| วิธี | SL = ขอบ OB ล่าง − 1 ถึง 2 pip |
| ทำไม | เผื่อ spread (0.2-0.5 pip) + noise (0.5-1 pip) |
| ถ้าไม่เผื่อ | noise จิ้ม → SL กินทั้งที่ OB ยังใช้ได้ |
| ตัวอย่าง | OB=4500 → SL=4498 (2 pip) TP=4520 RR=1:10 |

#### แบบ B: SL = ใต้ OB 0.5 ATR
| หัวข้อ | รายละเอียด |
|--------|-----------|
| ใช้ตอน | OB ธรรมดา, ไม่มี confluence ชัด |
| วิธี | SL = ขอบ OB ล่าง − (ATR × 0.5) |
| ATR คือ | ค่าเฉลี่ย volatility — ยิ่งสูง = ตลาดยิ่งผันผวน |
| ทำไม 0.5 ATR | เผื่อครึ่งหนึ่งของ volatility ปกติใน 1 แท่ง |
| ตัวอย่าง | OB=4500 ATR=30 pip → SL=4500−(30×0.5)=4485 |

#### แบบ C: SL = ใต้ FVG ล่างสุด
| หัวข้อ | รายละเอียด |
|--------|-----------|
| ใช้ตอน | OB + FVG overlap กัน |
| วิธี | SL = ขอบ FVG ล่าง − 1 pip |
| ทำไมใต้ FVG | FVG = แนวรับสุดท้าย — ถ้าทะลุ = โซนพัง |
| ตัวอย่าง | FVG=4498-4502 OB=4500 → SL=4497 |

### ข้อดี
- ✅ ขาดทุนจำกัด
- ✅ ไม่ต้องตัดสินใจตอนเครียด
- ✅ มีวินัย

### ข้อเสีย
- ❌ แคบไป → noise กิน
- ❌ กว้างไป → ขาดทุนมาก
- ❌ ไม่ตั้งเลย = หายนะ

### Use Case รวม
```
BUY GOLDmicro ที่ OB 4500 (แข็ง + FVG 4498-4502)
confluence = 2 → ใช้แบบ C
SL = ใต้ FVG ล่าง 1 pip = 4497
SL distance = 3 จุด
TP = 4520 (FVG ถัดไป)
RR = 20:3 = 1:7
```

## 🔹 TP = Take Profit

### ความหมาย
TP คือ **เส้นบอกว่า "ถึงเป้าแล้ว"** — เมื่อราคามาถึง ออเดอร์จะปิดอัตโนมัติเพื่อเก็บกำไร

### จุดประสงค์
1. **ป้องกันความโลภ** — ไม่รอจนกำไรหาย
2. **ทำให้แผนสมบูรณ์** — Entry + SL + TP ครบ
3. **วัดผลตัวเองได้** — รู้ว่าเทรดตามแผนหรือไม่

### วิธีใช้ — 3 แบบ

#### แบบ 1: OB/FVG ถัดไป
| หัวข้อ | รายละเอียด |
|--------|-----------|
| ใช้ตอน | มี OB/FVG อยู่ใน direction trade |
| วิธี | TP = ขอบโซน FVG/OB ที่ใกล้ที่สุด |
| ข้อดี | TP มีเหตุผลจากราคาจริง |
| ข้อเสีย | ถ้าโซนไกล → TP อาจไม่ถึง |
| ตัวอย่าง | BUY 4500 FVG ถัดไป 4520 → TP=4520 RR=20:3 |

#### แบบ 2: Previous High/Low
| หัวข้อ | รายละเอียด |
|--------|-----------|
| ใช้ตอน | OB อยู่ใน trending zone |
| วิธี | BUY → TP = Swing High ล่าสุด, SELL → TP = Swing Low ล่าสุด |
| ข้อดี | ตลาดชอบเทสจุดเดิม |
| ข้อเสีย | ใช้ได้ตอน trend เท่านั้น |
| ตัวอย่าง | BUY 4500 Swing High 4540 → TP=4540 |

#### แบบ 3: RR-based (1:2 / 1:3)
| หัวข้อ | รายละเอียด |
|--------|-----------|
| ใช้ตอน | ไม่มี zone อ้างอิง |
| วิธี | TP = SL distance × 2 หรือ × 3 |
| ข้อดี | ไม่ต้องคิด |
| ข้อเสีย | TP ไม่มีเหตุผลจากราคา |
| ตัวอย่าง | SL=5 จุด TP=10 จุด (1:2) หรือ TP=15 จุด (1:3) |

### ข้อดี
- ✅ เก็บกำไรตรงเป้า
- ✅ ไม่โลภ
- ✅ ตั้งอัตโนมัติ

### ข้อเสีย
- ❌ ตัน — ราคาอาจไปไกลกว่า TP
- ❌ เลือกผิดแบบ → TP ไม่ถึง
- ❌ RR-based ไม่มีเหตุผล

### Use Case รวม
```
BUY GOLDmicro ที่ OB 4500 SL=4497
เหนือขึ้นไปมี FVG ถัดไปที่ 4520
→ TP แบบ 1: TP = 4520
→ RR = (4520-4500) : (4500-4497) = 20:3 = 1:7 ✅
```

## คำศัพท์สำคัญ

### Pip / Point
- GOLDmicro: **1 pip = 0.10** (10 point), **1 point = 0.01**
- 4500.00 → 4500.10 = ขึ้น 1 pip

### ATR (Average True Range)
- "ค่าเฉลี่ยความผันผวนใน 1 แท่ง"
- ATR สูง = ตลาดป่วน → SL ต้องกว้าง
- ATR ต่ำ = ตลาดสงบ → SL แคบได้

### RR (Risk:Reward Ratio)
- "เสี่ยงเท่าไหร่ หวังเท่าไหร่"
- RR 1:2 = SL 1 จุด → TP 2 จุด
- RR 1:3 ขึ้นไป = trade คุณภาพดี

### Spread / Noise
- Spread = ส่วนต่างราคาซื้อ-ขาย (~0.2-0.5 pip)
- Noise = การแกว่งเล็กน้อย
- ต้องเผื่อ 1-2 pip ป้องกัน 2 อย่างนี้
