# 01 Candlestick Basics

## 🔹 แท่งเทียนคืออะไร? ทำไมต้องใช้?

ก่อนมีแท่งเทียน คนดูกราฟแบบ **line chart** — ต่อจุด Close ของแต่ละวัน ดูแค่ราคาปิด แต่ไม่รู้ว่า ระหว่างวันเกิดอะไรขึ้น

แท่งเทียน (candlestick) ถูกคิดค้นโดย **Munehisa Homma** พ่อค้าข้าวญี่ปุ่นในศตวรรษที่ 18 — เพื่อบันทึก **อารมณ์ตลาด** ในช่วงเวลานั้นๆ

**สิ่งที่แท่งเทียนบอก:**
1. ราคาเปิด-ปิด (Open-Close) — สรุปว่าช่วงนั้นใครชนะ
2. ราคาสูง-ต่ำ (High-Low) — ช่วงแดนของทั้ง 2 ฝ่าย
3. ขนาด Body — แรงของฝ่ายที่ชนะ
4. ความยาว Wick — แรงต้านของฝ่ายที่แพ้

> **หลักคิด:** ทุกแท่งเทียน = **การต่อสู้ระหว่าง Buy กับ Sell**  
> Body = ใครได้พื้นที่, Wick = โดนตีกลับ

---

## 🔹 ส่วนประกอบของแท่งเทียน

ลองนึกภาพ: ใน 1 ชั่วโมง (H1) ราคาทองคำขึ้นไปถึง 4505 ลงมา 4495 และปิดที่ 4502

```
         High = 4505 ──────┬──────
                           │
    Body ── Close = 4502 ──┤───
           Open  = 4498 ───┤
                           │
         Low  = 4495 ──────┴──────
```

| ส่วน | ค่าจากตัวอย่าง | แปลว่า |
|------|---------------|--------|
| **Open** | 4498 | ตอนเริ่มชั่วโมง — ราคาอยู่ที่ 4498 |
| **Close** | 4502 | ตอนจบชั่วโมง — ราคาอยู่ที่ 4502 |
| **High** | 4505 | ในชั่วโมงนั้น — ราคาขึ้นไปสูงสุดที่ 4505 |
| **Low** | 4495 | ในชั่วโมงนั้น — ราคาลงไปต่ำสุดที่ 4495 |
| **Body** | 4498 → 4502 | +4 จุด — Buyer ได้พื้นที่ |
| **Wick บน** | 4502 → 4505 | +3 จุด — Buyer พยายามขึ้นไป 4505 แต่โดน Seller ตีกลับ |
| **Wick ล่าง** | 4495 → 4498 | +3 จุด — Seller พยายามลงไป 4495 แต่โดน Buyer ดันกลับ |

> **ข้อสังเกต:** Wick = รอยที่ถูกตีกลับ — ถ้า wick ยาว แสดงว่าฝั่งตรงข้ามมีแรงต้านมาก

### Bullish vs Bearish

```
Bullish candle: Close > Open → Buyer ชนะ (เขียว/ขาว)
  - ราคาเริ่มที่ 4498 → ปิดที่ 4502
  - Body = 4 จุดเขียว
  - แปลว่า: ตลอดชั่วโมงนั้น Buyer ควบคุมราคาไว้ได้

Bearish candle: Close < Open → Seller ชนะ (แดง/ดำ)
  - ราคาเริ่มที่ 4502 → ปิดที่ 4498
  - Body = 4 จุดแดง
  - แปลว่า: Seller ควบคุมราคา
```

**Body ยิ่งยาว = ยิ่งแรง** — เช่น body เขียว 20 จุด vs body เขียว 2 จุด  
→ 20 จุด = Buyer มีกำลังมากกว่า 2 จุด 十倍

## 🔹 ประเภทของแท่งเทียน — ทำไมแต่ละแบบถึงมีความหมาย

### 1. Marubozu (ไม่มี wick)

```
Bullish Marubozu:
Open = Low → Close = High
           ↑
     ราคาเกิดที่ Low → วิ่งขึ้นไปเรื่อยๆ → จบที่ High
     ตลอดแท่งไม่มีแรงขายแม้แต่นิดเดียว

Bearish Marubozu:
Open = High → Close = Low
           ↑
     ราคาเกิดที่ High → ร่วงตลอด → จบที่ Low
     แรงซื้อ = 0
```

**ทำไม Marubozu ถึงสำคัญ:** มันบอกว่า **ฝั่งตรงข้ามไม่มีแรงต้านเลย**  
- Bullish Marubozu = Buyer มีอำนาจเบ็ดเสร็จ — ขึ้นต่อไปได้  
- Bearish Marubozu = Seller มีอำนาจเบ็ดเสร็จ — ลงต่อไปได้  

**วิธีใช้:** ถ้าเจอ Marubozu ในทิศทาง trend → เชื่อมั่นว่า trend ยังแข็ง  
ถ้าเจอ Marubozu กลับทิศ (เช่น Bearish Marubozu ใน uptrend) → ระวัง trend จบ

### 2. Doji (เปิด = ปิด)

Doji = Open ≈ Close — **Buyer กับ Seller เสมอกัน**

```
Doji:
      High ────┬───────
               │
      Open === │ === Close  ← ราคาเปิด = ปิด
               │
      Low ─────┴───────
```

**ทำไม Doji ถึงสำคัญ:** มันบอกว่า "ตลาดลังเล"

นึกภาพ: ราคาทองขึ้นมาสุดที่ 4520 โดนขายลงไป 4510 ซื้อกลับขึ้นมา 4520 อีก... สุดท้ายกลับมาปิดที่ 4515 (Open ≈ Close) → **เขาไม่รู้จะไปทางไหน** — ความลังเลนี้ = trend กำลังหมดแรง → มักเกิดก่อนกลับตัว

```
ตัวอย่าง:
  Uptrend มา 5 แท่ง → เจอ Doji
  → Buyer ยัง push ราคาขึ้นไป High แต่ Seller push กลับมาได้
  → สุดท้ายเสมอกัน
  → แปลว่า Buyer เริ่มอ่อน — trend อาจจบ
```

**รูปแบบ Doji:**

| แบบ | ลักษณะ | ทำไมถึงเกิด | แปลผล |
|-----|--------|------------|--------|
| Doji ปกติ | wick = ปกติ | Buyer/Seller เสมอกัน | ตลาดลังเล → รอดู |
| Long-legged | wick ยาวทั้ง 2 ข้าง | ตลาดรีดสูง-ต่ำมาก แต่สุดท้ายไม่ไปไหน | Volatile สูง → กำลังเปลี่ยน |
| Dragonfly | wick ล่างยาว ไม่มี wick บน | Seller push ลงแรง → Buyer ดันกลับมาเปิดปิดเท่าเดิม | แรงขายถูกปฏิเสธ → Bullish |
| Gravestone | wick บนยาว ไม่มี wick ล่าง | Buyer push ขึ้นแรง → Seller ดันกลับมาเปิดปิดเท่าเดิม | แรงซื้อถูกปฏิเสธ → Bearish |

### 3. Hammer / Shooting Star

**Hammer (ค้อน):** เกิดใน **downtrend** → wick ล่างยาว → ปิดสูง

```
Hammer:
     ┌────┐
     │    │──── Close
     │    │
     │    │──── Open
     └────┘
     │  wick ล่างยาว
     ▼
```

**ทำไม Hammer ถึง bullish:**  
ใน downtrend ที่ Seller เคยควบคุม — จู่ๆ มีแท่งที่ลงไปลึก (wick ล่าง) แล้วเด้งกลับมาปิดสูง  
→ Seller ดันลงไปเท่าไหร่? → Buyer ซื้อกลับมาได้หมด → Seller เริ่มหมดแรง

> เปรียบ: มวยที่โดนหมัดหนักๆ แต่ยังยืนได้ → กำลังจะโต้กลับ

**Shooting Star (ดาวตก):** เกิดใน **uptrend** → wick บนยาว → ปิดต่ำ

```
Shooting Star:
     ┌────┐──── High
     │    │
     │    │
     │    │
     └────┘
```

**ทำไม Shooting Star ถึง bearish:**  
ใน uptrend ที่ Buyer เคยควบคุม — จู่ๆ มีแท่งที่ขึ้นไปสูง (wick บน) แต่โดนขายกลับมาปิดต่ำ  
→ Buyer push ขึ้นไปเท่าไหร่ → Seller ขายกลับมาหมด → Buyer เริ่มหมดแรง

**ข้อแม้สำคัญ:** รูปแบบนี้ต้องเกิดใน trend ถึงจะมีความหมาย
- Hammer ใน sideways = ไม่มีความหมาย
- Shooting Star ใน downtrend แล้วเจอ = แค่ pullback ปกติ

### 4. Engulfing (กลืน)

**Bullish Engulfing:** แดง → เขียว body ใหญ่กว่ากลืนทั้งแท่ง

```
แท่ง 1 (แดง): ราคาปิดต่ำกว่าเปิด — Seller ชนะ  body = 5 จุด
แท่ง 2 (เขียว): ราคาปิดสูงกว่าเปิด — Buyer ชนะ  body = 12 จุด
               และ Open 2 < Close 1 + Close 2 > Open 1
               → body เขียวกลืน body แดงทั้งแท่ง
```

**ทำไม Engulfing ถึงแรง:** มันบอกว่า momentum เปลี่ยนทันที  
- เมื่อวาน Seller ยังชนะ (แท่งแดง)
- วันนี้ Buyer มาด้วยกำลังที่มากกว่า — กลืนพื้นที่ Seller ทั้งหมด
- → **Control shift:** จาก Seller → Buyer

**Bearish Engulfing:** ตรงข้าม — เขียว → แดงกลืน — Seller ยึดพื้นที่คืน

### 5. Pin Bar (wick ยาวข้างเดียว)

Pin Bar = ราคาวิ่งไปทางหนึ่ง → โดนปฏิเสธ → กลับมาปิด

```
Pin Bar Buy:
     ┌────┐
     │    │──── Open/Close
     │    │
     │    │
     └────┘──── wick ล่าง = 2-3× body
```

**ทำไม Pin Bar ถึงสำคัญ:**  
มันบอกว่า "ระดับนี้ ตลาดปฏิเสธราคา"

- Pin Bar Buy (wick ล่าง): ราคาลงไป Low → มีคนซื้อมหาศาลดันกลับ → "พื้นแข็ง"
- Pin Bar Sell (wick บน): ราคาขึ้นไป High → มีคนขายมหาศาลดันกลับ → "เพดานแข็ง"

**ข้อควรจำ:** Pin Bar ยิ่ง wick ยาว = แรงปฏิเสธยิ่งมาก  
แต่ต้องดูว่า rejection ครั้งนี้แข็งแรงพอไหม — ถ้า candle ถัดไปกลับไปทาง wick → rejection นั้นพัง

## 🔹 2 แท่งติดกันบอกอะไร?

แท่งเดียวอาจเป็น noise — 2 แท่งติดกันเริ่มมีนัยสำคัญเพราะมันแสดงลำดับของเหตุการณ์

| รูปแบบ | เกิดอะไรขึ้น | ความหมาย |
|--------|------------|----------|
| Bullish Engulfing | Seller ชนะ → Buyer กลับมาชนะด้วยแรงกว่า | กลับตัวขึ้น |
| Bearish Engulfing | Buyer ชนะ → Seller กลับมาชนะด้วยแรงกว่า | กลับตัวลง |
| Piercing | Seller ชนะแรง → Buyer ดีดกลับเกินครึ่งของ Seller | Buyer เริ่มมีแรง → รอ กลับตัวขึ้น |
| Dark Cloud Cover | Buyer ชนะแรง → Seller ร่วงเกินครึ่งของ Buyer | Seller เริ่มมีแรง → รอ กลับตัวลง |
| Tweezer Top | 2 แท่งสูงเท่ากันพอดี | แนวต้าน 2 ครั้งที่ระดับนี้ — แข็ง |
| Tweezer Bottom | 2 แท่งต่ำเท่ากันพอดี | แนวรับ 2 ครั้งที่ระดับนี้ — แข็ง |

## 🔹 สรุป: วิธีอ่านแท่งเทียน

```
แท่งใหญ่ (body กว้าง) = แนวโน้มแข็ง — ฝ่ายนั้นเชื่อมั่น
แท่งเล็ก (body แคบ)  = แนวโน้มอ่อน — ตลาดลังเล
Wick ยาว = โดนปฏิเสธ
  - Wick บน = Buyer โดนตีกลับ
  - Wick ล่าง = Seller โดนตีกลับ
ไม่มี wick = ฝั่งตรงข้ามไม่มีแรงต้าน
Doji หลัง trend = trend หมดแรง — เตรียมกลับ
```

**เทคนิค: อย่าตัดสินจาก 1 แท่ง — รอ 2-3 แท่งยืนยันเสมอ**  
ตัวอย่าง:  
- เห็น Shooting Star → รอ 2 แท่งถัดไป confirm → ลงต่อ → reversal จริง  
- เห็น Shooting Star → 2 แท่งถัดไปขึ้นต่อ → trend ยังแข็ง → Shooting Star = noise
