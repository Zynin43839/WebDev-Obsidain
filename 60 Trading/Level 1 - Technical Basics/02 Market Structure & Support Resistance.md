# 02 Market Structure & Support Resistance

---

## 🔹 Part 1: ทำไมต้องเข้าใจ Market Structure?

**Market Structure = โครงสร้างตลาด = แผนที่บอกว่า trend กำลังไปทางไหน**

การดู Market Structure คือการตอบคำถาม 3 ข้อ:
1. **ตลาดกำลังขึ้น ลง หรือ Sideway?**
2. **โครงสร้างนี้แข็งแรงแค่ไหน?**
3. **มันกำลังจะเปลี่ยนไหม?**

ข้อผิดพลาด #1 ของเทรดเดอร์มือใหม่: เห็นกราฟแท่งเขียว → เปิด Buy โดยไม่สนว่า trend โดยรวมเป็นยังไง  
→ ถ้า trend ใหญ่เป็นลง → Buy = ว่ายทวนน้ำ

> **หลักคิด: "Trade with the trend" = ดูโครงสร้างก่อน ถึงจะรู้ว่าอะไรคือ trend**

---

### Swing Points — จุดเปลี่ยนหลักของราคา

Swing High = จุดสูงสุดของรอบที่ราคากลับตัวลง  
Swing Low = จุดต่ำสุดของรอบที่ราคากลับตัวขึ้น

สังเกตว่า ราคาไม่เคยขึ้นหรือลงเป็นเส้นตรง — มันจะ swing (แกว่ง) ขึ้นลงสลับกัน แล้วค่อยๆ สร้าง Higher Highs ถ้าขึ้น หรือ Lower Highs ถ้าลง

```
Uptrend: HH (จุดสูงขึ้นเรื่อยๆ) + HL (จุดต่ำสูงขึ้นเรื่อยๆ)

         HH────┐
        /      \    HH────┐
       /        \  /      \
      /          \/        \
     /           HL        HL
    /           /
   /           /
  HL          /

1. HL แรก: จุดต่ำที่ Buyer ดันกลับ — ถ้าต่ำกว่านี้ = โครงสร้างพัง
2. HH แรก: จุดสูงใหม่ — Buyer push ขึ้นไปได้ไกลกว่าเดิม
3. HL ที่ 2: สูงกว่า HL แรก — แสดงว่า Buyer ไม่ปล่อยให้ราคาลงลึกเหมือนเดิม
4. HH ที่ 2: สูงกว่า HH แรก — Buyer push ได้ไกลขึ้นอีก

→ โครงสร้าง uptrend = จุดต่ำกับจุดสูงวิ่งขึ้นไปเรื่อยๆ
```

```
Downtrend: LH (จุดสูงต่ำลงเรื่อยๆ) + LL (จุดต่ำต่ำลงเรื่อยๆ)

  LH────┐
 /      \    LH────┐
/        \  /      \
          \/        \
          LL        LL
          /        /
         /        /
        LL      /

1. LH แรก: จุดสูงที่ Seller push กลับ — ถ้าสูงกว่านี้ = โครงสร้างพัง
2. LL แรก: จุดต่ำใหม่ — Seller push ลงไปได้ลึกกว่าเดิม
3. LH ที่ 2: ต่ำกว่า LH แรก — Seller ไม่ปล่อยให้ราคาขึ้นสูงเท่าเดิม
4. LL ที่ 2: ต่ำกว่า LL แรก — Seller push ลงไปลึกขึ้น

→ โครงสร้าง downtrend = จุดต่ำกับจุดสูงวิ่งลงไปเรื่อยๆ
```

---

### 3-Step Identify Trend — จำให้ติด

| Step | ดูอะไร | Uptrend | Downtrend | Sideway |
|------|--------|---------|-----------|---------|
| 1 | Swing High | Higher High (HH) | Lower High (LH) | Equal High (EQH) |
| 2 | Swing Low | Higher Low (HL) | Lower Low (LL) | Equal Low (EQL) |
| 3 | สรุป | HH + HL ✅ | LH + LL ✅ | อื่นๆ ⚠️ |

**วิธีใช้ในชีวิตจริง:**
1. เปิดกราฟ H1/D1
2. มองหาจุดสูงสุดและต่ำสุดของรอบล่าสุด 2-3 รอบ
3. เชื่อมจุดสูง → จุดสูง และจุดต่ำ → จุดต่ำ
4. อ่านผลจากตาราง

> **ตัวอย่าง:**  
> ดู D1 ทอง — Swing High: 4500 → 4550 → 4580 (HH)  
> Swing Low: 4480 → 4510 → 4530 (HL)  
> → Uptrend — หา Buy อย่างเดียว

---

### ทำไมโครงสร้างถึงสำคัญ — MSS (Market Structure Shift)

เมื่อไรก็ตามที่:
- **Uptrend** มี HL ถูกทำลาย → ราวลงต่ำกว่า HL ก่อนหน้า → **แนวโน้มกำลังเปลี่ยน**
- **Downtrend** มี LH ถูกทำลาย → ราวขึ้นสูงกว่า LH ก่อนหน้า → **แนวโน้มกำลังเปลี่ยน**

นี่คือ "**MSS — Market Structure Shift**" — จุดที่ Smart Money เปลี่ยนทิศ

> **HL = เส้นตายของ uptrend**  
> LH = เส้นตายของ downtrend

ถ้า HL หลุดใน uptrend = Buyer หมดแรง → อาจถึงเวลาเทรด Sell  
แต่ต้องรอ confirmation ก่อน — อย่าเทรดทันทีที่ HL หลุด เพราะอาจเป็นแค่ liquidity grab

## 🔹 Part 2: Support & Resistance (S/R)

### Support คืออะไร? ทำไมถึงเกิดขึ้น?

**Support = ระดับที่มีคำสั่งซื้อมากมายรออยู่ → ราวลงมาแล้วเด้งกลับ**

```
ราคา
 │
 │  ── Resistance ──  ← ตรงนี้มี Sell wall รอ
 │       ↓
 │       ↑  ← ราคาเด้งขึ้น — โดน Buy wall รับไว้
 │  ── Support ──  ← ตรงนี้มี Buy wall รอ
 │       ↓
 │       ↑  ← ราคาเด้งอีก
 │  ── Support ──
 ▼
```

**ทำไม Support ถึงรับราคาไว้ได้?**

1. **Swing Low ก่อนหน้า:** เทรดเดอร์จำได้ว่า "ครั้งที่แล้วราคาลงมาเท่านี้แล้วเด้ง" → รอบนี้ก็รอซื้อแถวนี้
2. **Order Block (OB):** Smart Money ทิ้งคำสั่งซื้อจำนวนมากไว้ที่ระดับนี้
3. **Fair Value Gap (FVG):** ราคาวิ่งผ่านเร็วไป — ยังมี "ฝั่งที่ยังไม่ถูกชำระ" (imbalance)
4. **Round Number:** 4500, 4600 — จิตวิทยาคนมักตั้ง Stop Buy/Sell ที่เลขกลมๆ
5. **Trendline:** หลายคนลากเส้นเชื่อม Low → Low → Low — ยิ่งคนใช้ร่วมกันยิ่งแข็ง

### Resistance คืออะไร? ทำไมถึงเกิดขึ้น?

**Resistance = ระดับที่มีคำสั่งขายมากมายรออยู่ → ราวขึ้นมาแล้วร่วงกลับ**

สาเหตุเดียวกันกับ Support แต่สลับฝั่ง — Swing High, OB ฝั่ง Sell, FVG, Round Number, Trendline

---

### วิธีวัดว่า S/R นั้นแข็งแค่ไหน

| ประเภท S/R | Reliability | ทำไมแข็ง/อ่อน |
|-------------|-------------|---------------|
| Swing Point (เคยมามาก่อน) | สูง | ราคาเคยเด้งตรงนี้ — "probattle zone" |
| Round Number | กลาง | จิตวิทยา — คนชอบตั้งคำสั่งที่เลขกลม |
| Trendline | กลาง | เชื่อมได้หลายแบบ — reliability ขึ้นกับจำนวนแต้ม |
| OB / FVG | สูงมาก | Smart Money ทิ้งร่องรอยไว้ |
| Moving Average | ต่ำ | Dynamic — เปลี่ยนทุกวัน |

**เทคนิคการหา S/R ที่น่าเชื่อถือ:**

1. **ดูประวัติว่าเคยกี่ครั้ง**
   - 1 ครั้ง = เริ่มสังเกต
   - 2 ครั้ง = น่าสนใจ
   - 3 ครั้งขึ้นไป = แข็งมาก

2. **S/R ที่เกิดจากหลายปัจจัยซ้อน**
   - 4500 เป็นทั้ง Round Number + Swing Low + OB → แข็งมาก
   - 4500 เป็นแค่ Round Number อย่างเดียว → ปานกลาง

3. **TF ยิ่งใหญ่ยิ่งแข็ง**
   - S/R บน D1 = แข็งกว่า S/R บน M15

---

### ทำไม S/R ยิ่งเทสเยอะยิ่งอ่อน?

นี่คือสิ่งที่มือใหม่สับสนบ่อย — "พอราคามาเทสที่ 4500 ครั้งที่ 5 ทำไมทะลุ?"

```
ครั้งที่ 1: ราคาแตะ 4500 → มีคนซื้อมหาศาล → รับไว้
ครั้งที่ 2: มีคนซื้อน้อยลงนิดหน่อย → ยังรับไว้
ครั้งที่ 3: เริ่มมีคนเปลี่ยนใจ → รับไว้ได้รอมรอม
ครั้งที่ 4: คนที่เคยซื้อกำไรแล้ว → ขายทำกำไร → รับไว้แทบไม่ได้
ครั้งที่ 5: Buy wall หมด → ทะลุ
```

**คำอธิบาย:** ทุกครั้งที่ Support ถูกราคาเทส = มีคนที่ซื้อแถว Support และขายทำกำไร  
→ Buy wall (คำสั่งซื้อ) ค่อยๆ หมดไป  
→ Support ยิ่งเทสยิ่งอ่อน — จนสุดท้ายทะลุ

## 🔹 Part 3: EQH / EQL (Equal Highs / Lows)

### EQH — ทำไมจุดสูงเท่ากันถึงเป็นแนวต้านแข็ง

```
4500 ──── EQH ────  ← ครั้งแรก ราคาขึ้นมาที่ 4500 แล้วร่วง
4500 ──── EQH ────  ← ครั้งที่ 2 ราคาขึ้นมาที่ 4500 แล้วร่วงอีก
```

**ทำไม EQH ถึงแข็ง?**  
เพราะเทรดเดอร์จำได้ว่า "4500 = จุดที่โดนตีกลับ 2 ครั้ง"  
→ ครั้งที่ 3 คนจะรอ Sell ที่ 4500  
→ Sell wall = ใหญ่ → ราคายิ่งขึ้นไปยาก

**ถ้าทะลุ EQH ได้ → ไปต่อแรง (Bullish Breakout)**

```
EQH = มีคนจำนวนมาก Short ไว้แถวนี้ (รอ Sell)
พอราคาทะลุ → คนพวกนี้ขาดทุน (Stop Loss ถูกกิน)
→ ราคาพุ่งต่อเพราะ:
   - Stop Loss คน Short กลายเป็น Buy order
   - คน Short โดนบังคับซื้อคืน (Short squeeze)
   - คนที่รอ Buy เห็นทะลุ → ตามซื้อ
→ 3 แรงซ้อน → ขึ้นแรง
```

**ถ้า EQH ไม่ทะลุ → Double Top → กลับตัวลง**  
คนที่เห็น EQH 2 ครั้ง → รอ Sell → ราคาร่วง

> **EQH/EQL = จุดที่ Smart Money ไปล่า Stop Loss**  
> Smart Money รู้ว่ามี Stop Loss มากมายอยู่ตรง EQH/EQL  
> มันอาจทะลุ EQH แค่หลอกๆ (liquidity grab) → แล้วกลับตัว  
> → อย่าเข้า breakout ทันที — รอ confirmation

### EQL — จุดต่ำเท่ากัน

```
4490 ──── EQL ────  ← ครั้งแรก ราคาลงมาที่ 4490 แล้วเด้ง
4490 ──── EQL ────  ← ครั้งที่ 2 ราคาลงมาที่ 4490 แล้วเด้ง
```

หลักการเดียวกันกับ EQH — มีคน Buy มากมายที่ 4490  
ถ้าทะลุ → Stop Loss โดนกิน → ขายตามกัน → ร่วงแรง  
ถ้าไม่ทะลุ → Double Bottom → กลับตัวขึ้น

## 🔹 Part 4: Role Reversal — ทำไม Support แตกแล้วกลายเป็น Resistance

### หลักการ

```
Support แตก → กลายเป็น Resistance
Resistance แตก → กลายเป็น Support
```

**ทำไมถึงเป็นแบบนั้น? — จิตวิทยา + Order Flow**

นึกภาพ: ทองมี Support ที่ 4500 (เด้ง 3 ครั้ง)

```
ครั้งที่ 1-3: ราคาลงมา 4500 → ซื้อ → เด้ง → กำไร — คนที่ซื้อแถวนี้ happy

ครั้งที่ 4: ราคาทะลุ 4500 ลงไป
→ คนที่ซื้อแถว 4500 = ขาดทุน → หวังว่า "เดี๋ยวก็กลับมา" (bag holder)
→ ราคาลงไปต่อ → คนเจ็บปวด → พอราคากลับมา 4500 = "โอกาสคืนทุน!"
→ รีบขาย (ขายเมื่อถึงจุดคุ้มทุน)
```

พอคนจำนวนมากขายพร้อมกัน → 4500 ที่เคยเป็น Support กลายเป็น Resistance

```
ครั้งที่ 5: ราคากลับมา 4500
→ คนที่ซื้อแถว 4500 ยังขาดทุนอยู่ → รีบขายคืนทุน
→ Sell wall = ใหญ่ → ราคาเด้งลง
→ 4500 = Resistance แล้ว
```

> **Role Reversal = ทุกแนวรับที่ทะลุ = แนวต้านในอนาคต**  
> ทุกแนวต้านที่ทะลุ = แนวรับในอนาคต

### วิธีเทรด Role Reversal

```
Support 4500 แตก → ลงไป 4470 → กลับมาเทส 4500
→ ตอนนี้ 4500 = Resistance (เปลี่ยนบทบาทแล้ว)
→ รอ Sell ที่ 4500
→ SL = 4505 (เหนือ Resistance นิดหน่อย)
→ TP = 4470 (Low เก่า) หรือต่ำกว่า
```

## 🔹 Part 5: Trendline — ทำไมต้องลากถูกวิธี

### วิธีลากที่ถูก

```
Uptrend: เชื่อม Swing Low → Swing Low → Swing Low
  = Higher Lows

Downtrend: เชื่อม Swing High → Swing High → Swing High
  = Lower Highs
```

**ข้อผิดพลาดมือใหม่:** ลาก trendline ผ่าน body แท่งเทียน → ลากวกไปมา → ไม่มีความหมาย

### กฎ 3 ข้อของ Trendline

1. **แตะอย่างน้อย 2 จุด = เริ่มมีนัยสำคัญ**
   - 2 จุด = เส้นตรงตามนิยาม
   - 3 จุด = แนวโน้มแข็ง — ราคาเคารพเส้นนี้
   - ยิ่งแตะเยอะ = ยิ่งแข็ง

2. **45° = สมดุล**
   - Trendline ชันเกินไป = แนวโน้มแรงแต่เปราะ — ทะลุง่าย
   - Trendline ราบ = ตลาด sideways
   - 45° = ทองคำแท่ง — trend ที่เดินสบายๆ

3. **Trendline Break ≠ กลับตัวเสมอไป**
   - บางที trendline แค่ flatten
   - ต้องดูร่วมกับ Structure break (MSS)

### Retest หลัง Trendline Break

```
trendline แตก → กลับมาเทส → ถ้าไม่กลับ → trendline broken = จริง
trendline แตก → กลับมาเทส → กลับเข้า trend → fakeout → trendline ยังใช้ได้
```

## 🔹 สรุป: Market Structure + S/R Workflow

```
Step 1: ระบุ Trend ด้วย Swing Points
  - HH + HL = Uptrend → มองหา Buy
  - LH + LL = Downtrend → มองหา Sell
  - อื่นๆ = Sideway → ข้าม หรือ ระวัง

Step 2: หา S/R ที่แข็ง
  - Swing Points (ประวัติเคยมามาก่อน)
  - OB / FVG (จาก SMC)
  - Round Numbers (4500, 4600)
  - เทสกี่ครั้ง? ยิ่งเทสเยอะยิ่งอ่อน

Step 3: หา EQH/EQL
  - จุดที่ liquidity สะสม
  - จุดที่อาจมี liquidity grab
  - จุดที่อาจเป็น Double Top/Bottom

Step 4: รอราคามาเทส S/R
  - อย่าเข้า S/R ล่วงหน้า — รอให้ราคามาจริง
  - เคารพ S/R → รอ entry
  - ไม่เคารพ → ข้าม

Step 5: Role Reversal
  - S/R แตก → กลับมาเทส → เปลี่ยนบทบาท → เทรดทางตรงข้าม

Step 6: Entry ตาม SMC Concepts
  - Entry A (Limit), B (Reversal), C (Breakout)
  - SL & TP ตาม Concept
```
