# Confluence 3 — Concept 6C: Divergence + Order Block + Market Structure Shift

---

## 🔹 ความหมาย

เมื่อทั้ง 3 องค์ประกอบ (Divergence, Order Block, Market Structure Shift) อยู่ในทิศเดียวกัน = **สัญญาณที่แข็งแกร่งที่สุดใน SMC ที่เรามี**

```
Divergence = "กำลังจะเปลี่ยน" (early signal)   → reliability 40%
+ Market Structure Shift = "กำลังเปลี่ยน" (confirmation)   → reliability 70%
+ Order Block = "เปลี่ยนแล้ว" (entry zone)     → reliability 90%+
```

---

## 🔹 จุดประสงค์

1. **สร้างชุดสัญญาณที่ reliability สูงที่สุด (90%+)** — trade น้อยลง แต่ win rate สูงขึ้น
2. **มี Workflow ที่ชัดเจน** — divergence → Market Structure Shift → Order Block → Entry — ไม่ใช่การเดา
3. **กัน FOMO** — ถ้าไม่ครบ 3 องค์ประกอบ = ไม่เข้า (แม้ divergence จะสวยแค่ไหน)

---

## 🔹 วิธีใช้: 3-Step Workflow

### Step 1: Divergence ปรากฏ (Early Warning)

```
เช็ค RSI → พบ Regular Divergence
→ ตลาดกำลังเตือนว่า trend นี้กำลังหมดแรง
→ ยังไม่เข้า — แค่เริ่มหา Order Block ฝั่งตรงข้าม
→ Mark จุดที่ Divergence เริ่มไว้

สิ่งที่ไม่ควรทำ:
✗ เปิด Sell ทันทีที่เห็น Bearish Divergence
✗ ทิ้ง Order Block เดิมแล้วกลับฝั่ง

สิ่งที่ควรทำ:
✓ จดไว้ในหัวว่า "trend นี้อาจจบ"
✓ เตรียมหา Order Block ฝั่งตรงข้าม
```

### Step 2: Market Structure Shift ยืนยัน (Confirmation)

```
ราคาทำลาย key level (Higher Low/Lower High) → External Change of Character
→ แนวโน้มเปลี่ยนแล้วจริง
→ Order Block ฝั่งเก่า = โมฆะ (ใช้ไม่ได้อีก)
→ หา Order Block ฝั่งตรงข้าม

สิ่งที่ไม่ควรทำ:
✗ เข้าทันทีที่เห็น Market Structure Shift — ยัง ไม่มี Order Block
✗ ใช้ Order Block เก่าใน trend ใหม่

สิ่งที่ควรทำ:
✓ เปิดหา Order Block ฝั่งตรงข้าม (OB ของ impulse ที่ทำให้ Market Structure Shift)
✓ เช็ค Confluence เตรียมไว้
```

### Step 3: รีเทสต์ Order Block (Entry Zone)

```
ราคามารีเทสต์ Order Block ฝั่งใหม่
→ Divergence (1) + Market Structure Shift (1) + Order Block (1) = Confluence 3 🎯
→ เข้าเทรดตามสถานการณ์ (A/B/C)

สิ่งที่ไม่ควรทำ:
✗ เข้าแบบ Limit Order (A) ถ้า Market Structure Shift แบบช้าๆ
✗ วาง Stop Loss แคบเกิน — ตลาดเพิ่งเปลี่ยน trend → อาจ oscillation

สิ่งที่ควรทำ:
✓ เลือก Entry Method ให้เหมาะกับความแรงของ Market Structure Shift
✓ ใช้ TP 2 ระดับ — Market Structure Shift มักไปไกล
```

### Visual Flow Timeline

```
เวลา:     t1 ─── t2 ─── t3 ─── t4 ─── t5 ─── t6 ─── t7
ราคา:     Higher High ─── Higher Low ─── Higher High ─── Higher Low ─── Lower Low ─── Lower High ─── Lower Low
RSI:      peak ─── กลาง ─── lower peak ─── cross 50 ───
เหตุการณ์: Divergence ───────── Market Structure Shift ──────── Order Block entry ────
                                 ↑           ↑
                             เกิด Change of Character    รีเทสต์ Order Block → เข้า
```

## 🔹 ข้อควรระวังในการใช้ Confluence 3

### 1. Divergence ที่ "ใช้ได้" จริงๆ

```
เช็ค Divergence ว่ามีน้ำหนัก:
[ ] RSI > 70 (bearish) หรือ RSI < 30 (bullish) ก่อน Divergence?
[ ] Swing point บน RSI ชัดเจน (ไม่ใช่แค่ย่อ)?
[ ] RSI ไม่ sideway อยู่แถว 40-60?
→ ถ้าไม่ครบ → reliability Divergence ลดลง
```

### 2. Market Structure Shift ต้องเป็น External

```
Internal Change of Character + Divergence = trend เดิมยังอยู่ → อย่าเพิ่งกลับฝั่ง
External Change of Character + Divergence = trend เปลี่ยน → กลับฝั่งได้
```

### 3. Order Block ต้องเป็น "OB ที่ชัดเจน"

```
Order Block ที่ดี:
- candle ใหญ่ (impulse ชัด)
- ราคาเคยรีเทสต์แล้วเด้ง (มีประวัติ)
- ไม่มี wick ทะลุเยอะ
```

## 🔹 ข้อดี

- ✅ Confluence 3 = reliability 90%+
- ✅ Workflow ชัด: divergence → Market Structure Shift → Order Block → Entry
- ✅ trade น้อยลง quality สูงขึ้น
- ✅ มีวินัย — ไม่เข้าเพราะ divergence อย่างเดียว

## 🔹 ข้อเสีย

- ❌ หาได้ยาก — divergence + Market Structure Shift + Order Block ครบ = ไม่กี่ครั้งต่อวัน (หรือต่อสัปดาห์)
- ❌ ถ้าครบ 3 แล้วยังผิด → เสียความมั่นใจ
- ❌ ต้องใช้เวลารอ — divergence → Market Structure Shift ใช้เวลา 5-10 candle

## 🔹 Use Case เต็ม

```
XAUUSD M15:

t1: ราคาทำ Higher High ที่ 4720, RSI = 78
t2: ราคาย่อ Higher Low 4690, RSI = 60
t3: ราคาทำ Higher High ที่ 4730, RSI = 70 (Lower High) → Bearish Regular Divergence! ⚠️

สรุป: Divergence แล้ว → เตรียมหา Order Block ขาย

t4: ราคาลงทำ Higher Low = 4680 (ยังสูงกว่า 4690? ไม่ → 4690 โดน break)
t5: ราคาลงต่อ Lower Low = 4650 → External Change of Character → Market Structure Shift SELL ✅

สรุป: Market Structure Shift แล้ว → Order Block BUY เก่าเป็นโมฆะ → หา Order Block ขาย

t6: ราคารีเทสต์ Order Block ที่ 4670 (Order Block ก่อน impulse ลง)
→ Divergence (1) + Market Structure Shift (1) + Order Block (1) = Confluence 3 🎯

Action:
Entry: Sell ที่ 4670 (Entry B — รอ bearish candle ยืนยัน)
Stop Loss: 4675 (Order Block + 5 pip — เผื่อ oscillation)
TP1: 4630 (FVG ถัดไป) = 40 pip
TP2: 4600 (Swing Low ก่อนหน้า) = 70 pip

Risk to Reward Ratio:
- เสี่ยง 5 จุด ได้ 40 จุด (TP1) = 1:8
- หรือ 70 จุด (TP2) = 1:14
```

## 🔹 Checklist ก่อนเข้า Confluence 3 Trade

```
☐ Divergence: RSI หรือ Indicator อื่นแสดง Divergence?

☐ Market Structure Shift Type: External Change of Character (ไม่ใช่ Internal)?
☐ Order Block: หา Order Block ฝั่งตรงข้ามเจอ?
☐ Order Block Quality: แท่งใหญ่ + impulse ชัด?
☐ Confluence: Divergence + Market Structure Shift + Order Block = 3?
☐ Entry: เลือก Method (A/B/C) ตามความแรง Market Structure Shift?
☐ Stop Loss: เผื่อ oscillation trend ใหม่?
☐ Take Profit: ตั้ง 2 ระดับ? (TP1 = FVG ถัดไป, TP2 = Swing Point)
☐ Risk to Reward Ratio: >= 1:3?
```
