# 📚 Trading Fundamentals Curriculum

> **เส้นทางการเรียนรู้จาก 0 สู่ SMC Trading** — เรียนตามลำดับ อย่าข้าม ข้ามทีไร = ขาดทุนทีนั้น

---

## 🔹 ภาพรวม Curriculum

```
Level 0: Mindset & Foundations
Level 1: Technical Analysis Basics
Level 2: Smart Money Concepts (SMC)
Level 3: Divergence & Confluence
Level 4: Psychology & Risk Management
Level 5: Live Trading & Optimization
```

**เวลาโดยประมาณ:** 3-6 เดือน (ถ้าทำทุกวัน)
**เวลาในแต่ละ Level:** ไม่เท่ากัน — Level 2 ใช้เวลามากที่สุด

---

## 🔹 Level 0: Mindset & Foundations (1-2 สัปดาห์)

### เป้าหมาย
ปรับ mindset และเข้าใจว่าตลาดการเงินทำงานยังไง ก่อนแตะกราฟ

### หัวข้อ

#### 0.1 ตลาดการเงินเบื้องต้น
- [ ] ตลาด Forex คืออะไร? ( decentralized, OTC )
- [ ] คู่เงินหลัก: XAUUSD (Gold), EURUSD, USDJPY
- [ ] เวลาตลาด: London, NY, Asia Session
- [ ] Spread, Pip, Point, Lot Size
- [ ] Leverage คืออะไร? ข้อดี-ข้อเสีย

#### 0.2 Mindset พื้นฐาน
- [ ] ตลาดไม่มี "ถูก-ผิด" — มีแต่ "probability"
- [ ] ขาดทุน = ส่วนหนึ่งของเกม (accept it)
- [ ] "รวยเร็ว" = "เจ๊งเร็ว" — ไม่มี例外
- [ ] เทรดเดอร์ vs นักพนัน — เส้นแบ่งอยู่ตรงไหน
- [ ] **อ่านต่อ:** SMC Concept 3 - Discipline Win

#### 0.3 คำศัพท์พื้นฐานที่ต้องรู้
- [ ] Bullish / Bearish
- [ ] Long / Short
- [ ] Support / Resistance
- [ ] Trend / Sideway
- [ ] Volatility / Liquidity

### การบ้าน Level 0
- ✅ เปิด Demo Account (XM, IC Markets, หรือ Exness)
- ✅ ดูกราฟวันละ 15 นาที โดยไม่เทรด — แค่ทำความคุ้นเคย
- ✅ จำ Session Times ให้ได้

## 🔹 Level 1: Technical Analysis Basics (3-4 สัปดาห์)

### เป้าหมาย
อ่านกราฟเป็น, รู้จัก Price Action เบื้องต้น, เข้าใจโครงสร้างตลาด

### หัวข้อ

#### 1.1 Candlestick Basics
- [ ] ส่วนประกอบของแท่งเทียน: Open, High, Low, Close
- [ ] Bullish vs Bearish candle
- [ ] Body, Wick, Doji
- [ ] แท่งเทียนสำคัญ: Engulfing, Pin Bar, Inside Bar, Hammer, Shooting Star
- [ ] 2 แท่งติดกันบอกอะไร?
- [ ] **อ่านต่อ:** [[01 Candlestick Basics]]

#### 1.2 Market Structure (โครงสร้างตลาด)
- [ ] Higher High (HH) / Higher Low (HL)
- [ ] Lower High (LH) / Lower Low (LL)
- [ ] Uptrend vs Downtrend vs Sideway
- [ ] Swing High / Swing Low
- [ ] **อ่านต่อ:** SMC Concept 0 - Foundations (Part 2)

#### 1.3 Support & Resistance
- [ ] S/R จาก Swing Points
- [ ] S/R จาก Round Numbers (4500, 4600)
- [ ] S/R จาก Trendlines
- [ ] Role Reversal (Support → Resistance, Resistance → Support)
- [ ] EQH / EQL (Equal Highs/Lows)
- [ ] **อ่านต่อ:** [[02 Market Structure & Support Resistance]]

#### 1.4 Price Action เบื้องต้น
- [ ] Breakout — ทะลุแนวรับ/แนวต้าน
- [ ] Pullback — ย่อตัวก่อนไปต่อ
- [ ] Retest — กลับมาเทสแนวรับ/แนวต้าน
- [ ] Fakeout / False Break — ทะลุแล้วกลับ

#### 1.5 Timeframe Analysis
- [ ] Higher Timeframe (HTF) = เทรนใหญ่ (H4, D1)
- [ ] Lower Timeframe (LTF) = entry (M15, H1)
- [ ] วิธีเทียบ TF: HTF direction → LTF entry
- [ ] Multiple Timeframe Analysis (MTF)
- [ ] **อ่านต่อ:** [[03 Price Action & Timeframe Analysis]]

### การบ้าน Level 1
- ✅ ทำความรู้จัก "Order Block" ผ่าน Concept 1 (อ่านเฉยๆ ยังไม่ต้องเข้า)
- ✅ หา HH/HL/LH/LL บนกราฟจริงวันละ 5 ตัวอย่าง
- ✅ จด "โครงสร้างตลาด" ของวันนั้น: uptrend/downtrend/sideway

## 🔹 Level 2: Smart Money Concepts (SMC) — 6-8 สัปดาห์

### เป้าหมาย
เข้าใจ SMC ครบทุกองค์ประกอบ — ตั้งแต่ Accumulation → Distribution → Entry

### หัวข้อ (เรียงตามลำดับ)

#### 2.1 SMC Foundations
- [ ] Smart Money คือใคร? ทำไมถึงสำคัญ?
- [ ] Institutional vs Retail
- [ ] Iceberg Orders & Absorption
- [ ] Accumulation → Manipulation → Distribution → Markdown
- [ ] **อ่านต่อ:** SMC Concept 0 - Foundations (Part 1)

#### 2.2 Liquidity
- [ ] Buy-Side Liquidity (BSL)
- [ ] Sell-Side Liquidity (SSL)
- [ ] Liquidity Grab / Stop Hunt
- [ ] วิธีแยก Grab vs Breakout จริง
- [ ] **อ่านต่อ:** SMC Concept 0 - Foundations (Part 3)

#### 2.3 Order Block (OB) — หัวใจของ SMC
- [ ] Order Block คืออะไร?
- [ ] วิธี identify OB (candle ก่อน impulse)
- [ ] OB ที่แข็ง vs OB ที่อ่อน
- [ ] **อ่านต่อ:** SMC Concept 1 - Entry A B C
- [ ] **อ่านต่อ:** SMC Concept 2 - SL TP Placement

#### 2.4 Fair Value Gap (FVG) / Imbalance
- [ ] FVG เกิดยังไง? (3-candle rule)
- [ ] ประเภท: Bullish FVG / Bearish FVG
- [ ] FVG ตาม Timeframe (Micro → Major)
- [ ] FVG + OB = Confluence 2
- [ ] **อ่านต่อ:** SMC Concept 0 - Foundations (Part 4)

#### 2.5 Market Structure Shift (MSS / CHoCH)
- [ ] MSS = โครงสร้างตลาดเปลี่ยนทิศ
- [ ] Pullback vs MSS — แยกให้ออก
- [ ] Internal CHoCH vs External CHoCH
- [ ] 3 ตัวช่วยยืนยัน: Volume, RSI, Candlestick Pattern
- [ ] Role Reversal (OB/S/R สลับบทบาท)
- [ ] **อ่านต่อ:** SMC Concept 4+5 MSS

#### 2.6 Entry Methods A/B/C
- [ ] A: Limit Order (รอที่ OB)
- [ ] B: Retest Confirmation (รอเด้งก่อน)
- [ ] C: Breakout Reclaim (false break)
- [ ] ตารางเลือกใช้ตามสถานการณ์
- [ ] **อ่านต่อ:** SMC Concept 1 - Entry A B C

#### 2.7 SL/TP Placement
- [ ] SL 3 แบบ: ใต้ OB, ใต้ OB+ATR, ใต้ FVG
- [ ] TP 3 แบบ: OB/FVG ถัดไป, Swing High/Low, RR-based
- [ ] RR (Risk:Reward) — หัวใจของการอยู่รอด
- [ ] **อ่านต่อ:** SMC Concept 2 - SL TP Placement

### การบ้าน Level 2
- ✅ วาด Market Structure + OB + FVG บนกราฟวันละ 3 ครั้ง
- ✅ เขียน "SMC Analysis" ของวันลงสมุด: trend → OB → FVG → liquidity
- ✅ ยังไม่เข้าเทรดจริง — แค่ Backtest บน Demo
- ✅ ทบทวน Concept 3 (Discipline Win) ทุกวันก่อนเปิดกราฟ

## 🔹 Level 3: Divergence & Confluence (2-3 สัปดาห์)

### เป้าหมาย
เพิ่มตัวกรองเพื่อยืนยันสัญญาณ — ลด false signal, เพิ่ม win rate

### หัวข้อ

#### 3.1 RSI Divergence
- [ ] RSI คืออะไร? (Relative Strength Index)
- [ ] วิธีลาก Trendline บน RSI
- [ ] Regular Divergence: Bullish / Bearish
- [ ] Hidden Divergence: Bullish / Bearish
- [ ] **อ่านต่อ:** SMC Concept 6A - วิธีลาก Trendline บน RSI
- [ ] **อ่านต่อ:** SMC Concept 6B - Trend Continuation Signal

#### 3.2 5 Indicators จับ Divergence
- [ ] MACD — Histogram Divergence (H1+)
- [ ] Stochastic — เร็วที่สุด (M15)
- [ ] CCI — จับ Turning Point (H1+)
- [ ] OBV — Volume ก่อนราคา (ทุก TF)
- [ ] Awesome Oscillator — ภาพชัด (M15)
- [ ] วิธีเลือก Indicator ตาม Timeframe
- [ ] **อ่านต่อ:** SMC Concept 6A-2 - Indicator จับ Divergence

#### 3.3 Confluence 3 — จุดสูงสุดของ SMC
- [ ] นิยาม: Divergence + MSS + OB
- [ ] Workflow 3-Step: Divergence ปรากฏ → MSS ยืนยัน → OB entry
- [ ] Reliability: 40% → 70% → 90%+
- [ ] Checklist ก่อนเข้า Confluence 3 Trade
- [ ] **อ่านต่อ:** SMC Concept 6C - Confluence 3

### การบ้าน Level 3
- ✅ หา Divergence บนกราฟวันละ 3 ครั้ง (เขียนลงสมุด)
- ✅ หา Confluence 3 trade — 1 ครั้งต่อวัน (ดูว่ามีไหม)
- ✅ Backtest 10-20 trades บน Demo โดยใช้ Checklist

## 🔹 Level 4: Psychology & Risk Management (2-3 สัปดาห์)

### เป้าหมาย
อยู่รอดในระยะยาว — 90% ของเทรดเดอร์เจ๊งเพราะ psychology ไม่ใช่ analysis

### หัวข้อ

#### 4.1 Risk Management
- [ ] กฎ 1-2% — เสี่ยงไม่เกิน 1-2% ของทุนต่อ trade
- [ ] Position Sizing: ใช้ lot size เท่าไหร่?
- [ ] Max Drawdown: หยุดเทรดเมื่อถึงจุดนี้
- [ ] Risk to Reward Ratio: 1:2, 1:3 — ควรใช้เท่าไหร่?
- [ ] Correlation Risk: อย่าเทรดคู่ที่ correlated พร้อมกัน
- [ ] **อ่านต่อ:** [[01 Risk Management & Position Sizing]]

#### 4.2 Trading Psychology
- [ ] FOMO (Fear Of Missing Out) — ตัวร้ายอันดับ 1
- [ ] Revenge Trading — ยิ่งแก้อึ่งแพ้
- [ ] Over-trading — ยิ่งเทรดเยอะ ยิ่งเสีย spread
- [ ] Analysis Paralysis — Indicator มากไป = เลือกไม่ถูก
- [ ] "ต้องชนะทุกครั้ง" — mindset ที่พัง
- [ ] **อ่านต่อ:** [[02 Trading Psychology & Journal]]
- [ ] **อ่านต่อ:** SMC Concept 3 - Discipline Win

#### 4.3 Pre-Trade Ritual
- [ ] Pre-Trade Checklist ก่อนกด Enter ทุกครั้ง
- [ ] **อ่านต่อ:** Pre-Trade Checklist
- [ ] Trading Journal — จดทุกเทรด (entry, SL, TP, เหตุผล, อารมณ์)
- [ ] Weekly Review: ดูสถิติ 1 ครั้งต่อสัปดาห์
- [ ] Post-Trade Analysis: ชนะ — ทำไมถึงชนะ? แพ้ — ทำไมถึงแพ้?

### การบ้าน Level 4
- ✅ ทำ Trading Journal อย่างน้อย 20 trades
- ✅ ใช้ Pre-Trade Checklist ทุกครั้งก่อนเข้า — ไม่มี Exception
- ✅ คำนวณ Position Size ก่อนเข้าเทรดทุกครั้ง
- ✅ หยุดเทรด 1 วันถ้าขาดทุนติดต่อกัน 2 ครั้ง (2-day loss rule)

## 🔹 Level 5: Live Trading & Optimization (Continuous)

### เป้าหมาย
เอาระบบที่เรียนรู้ไปใช้จริง + ปรับปรุงอย่างต่อเนื่อง

### ขั้นตอน

#### 5.1 Demo Trading Phase (30-50 trades)
- [ ] เทรดบน Demo ด้วยระบบ SMC เต็มรูปแบบ
- [ ] ใช้ Pre-Trade Checklist ทุกครั้ง
- [ ] จด Trading Journal
- [ ] หา Win Rate + Avg RR ของตัวเอง
- [ ] ปรับปรุงระบบตามสถิติ
- [ ] **เงื่อนไขผ่าน:** Win Rate ≥ 40% + Avg RR ≥ 1:2 + 30 trades

#### 5.2 Micro Live Trading (20-30 trades)
- [ ] ใช้เงินจริงแต่很小的 lot (0.01)
- [ ] จุดประสงค์: จัดการอารมณ์ตอนเงินจริง
- [ ] จดทุกเทรด + อารมณ์ตอนนั้น
- [ ] **เงื่อนไขผ่าน:** ทำตาม Checklist ได้ 100%, ไม่ revenge trade

#### 5.3 Scaling Up
- [ ] เพิ่ม lot size ตามระบบ (ไม่ใช่ตามอารมณ์)
- [ ] ตั้ง Target: % return ต่อเดือน (realistic — 5-10%)
- [ ] Technical Analysis Review ทุกสัปดาห์
- [ ] ปรับปรุงระบบตลอดเวลา — SMC ไม่ตายตัว

### การบ้าน Level 5
- ✅ Live Trading Journal
- ✅ รีวิว Performance ทุกสัปดาห์
- ✅ ปรับ Position Sizing ตาม Equity Curve

## 🔹 ข้อควรจำก่อนเริ่มเรียน

### 3 อย่า

1. **อย่าเร่ง** — Level 0-1 ข้ามไม่ได้ = เหมือนสร้างตึกไม่ลงเสา
2. **อย่าเชื่อทุกอย่าง** — SMC เป็นแนวคิด ไม่ใช่กฏตายตัว ถ้าตลาดไม่ respect OB → มีอะไรผิดปกติ
3. **อย่าเอาเงินจริงมาเล่นก่อนพร้อม** — Demo จนกว่า Checklist จะเป็น automatic

### 3 ต้อง

1. **ต้องจด** — Trading Journal บันทึกทุกเทรด (แม้แต่เทรดที่ข้าม)
2. **ต้องทบทวน** — Concept 3 (Discipline) อ่านซ้ำทุกสัปดาห์
3. **ต้องอดทน** — 3-6 เดือนในการเป็น profitable trader = เร็วมากแล้ว
