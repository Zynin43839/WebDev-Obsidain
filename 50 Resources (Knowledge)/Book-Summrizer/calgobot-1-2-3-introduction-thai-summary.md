# สรุปหนังสือ 1-2-3 Trading Signal — ทุกบท (ภาษาไทย)

**ผู้เขียน**: Mark Crisp | **Source**: stressfreetrading.com

---

## บทที่ 1: 1-2-3 Pattern คืออะไร?

**1-2-3 Pattern (รูปแบบ 1-2-3)** คือรูปแบบกราฟที่เกิดขึ้นบ่อยและใช้ระบุจุดกลับตัวของราคาได้ง่ายมาก มี 3 จุดหลัก:

### 1-2-3 Buy Pattern (รูปแบบซื้อ)
- **Point 1 (จุดที่ 1)**: ราคาต่ำสุด — bottom ของแนวโน้มขาลง
- **Point 2 (จุดที่ 2)**: ราคาเด้งกลับขึ้นมาเล็กน้อย — correction (การปรับฐาน)
- **Point 3 (จุดที่ 3)**: ราคากลับลงมาทดสอบจุดต่ำสุดเดิม **แต่ไม่ต่ำกว่า Point 1** — retest (การทดสอบใหม่)

ถ้า Point 3 ต่ำกว่า Pattern นี้ใช้ไม่ได้ — ให้ตัดทิ้งแล้วหาใหม่

**สัญญาณซื้อ**: ซื้อเมื่อราคา break out เหนือจุด Point 2

### 1-2-3 Sell Pattern (รูปแบบขาย)
กลับกันทั้งหมด — เกิดที่จุดสูงสุดของแนวโน้มขาขึ้น

### ความแม่นยำ
- รูปแบบนี้แม่นยำประมาณ **70%** ในการทำนายทิศทางราคา
- แต่ไม่ใช่ 100% (ผู้เขียนจะอธิบายเหตุผลในบทที่ 4)

## บทที่ 2: 1-2-3 Pattern เกิดที่ไหนบ้าง?

### ใช้ได้กับทุก Time Frame
- **Daily chart**: 2-8 สัปดาห์
- **Weekly chart**: 4-16 สัปดาห์
- **Monthly chart**: 2-12 เดือน (และอาจนานกว่านั้น)

**กฎทั่วไป**: ยิ่งมีแท่งเทียน (bars) เกี่ยวข้องมากเท่าไหร่ ขนาดการเคลื่อนไหวของราคาก็ยิ่งใหญ่เท่านั้น — 1-2-3 บน monthly chart จะให้สัญญาณที่ใหญ่กว่า weekly chart ซึ่งใหญ่กว่า daily chart

### ใช้ได้กับทุกประเภทกราฟ
- Bar chart, Line chart, Candlestick chart, Renko chart, Point & Figure chart
- ทุกประเภทกราฟแสดงรูปแบบ 1-2-3 ได้เหมือนกันหมด

### ใช้ได้กับทุกตลาด
- หุ้น, สินค้าโภคภัณฑ์ (commodities), ฟิวเจอร์ส (futures), ค่าเงิน (forex)
- ทั้งฝั่งซื้อ (long) และขาย (short)

## บทที่ 3: เทคนิค Entry "แอบเข้า"

แทนที่จะซื้อเมื่อราคา break out จุด Point 2 พอดี (ซึ่งเป็นจุดที่คนส่วนใหญ่แห่กันเข้า) ให้ **วางออเดอร์ซื้อไว้ต่ำกว่าจุด breakout เล็กน้อย** เพื่อเข้าตลาดก่อนฝูงชน

### เหตุผล
- เมื่อราคา break จุด Point 2 จะมีออเดอร์ซื้อจำนวนมากเข้าพร้อมกัน → ทำให้เกิด slippage (ราคาซื้อได้สูงกว่าที่ต้องการ)
- การเข้าก่อนหน้าจะช่วยลด slippage และมั่นใจว่าได้เข้าทันทีเมื่อเกิดการ breakout

### วิธีทำ
- **ฝั่งซื้อ**: วาง limit buy ต่ำกว่าจุด Point 2 เล็กน้อย
- **ฝั่งขาย**: วาง limit sell สูงกว่าจุด Point 2 เล็กน้อย
- จำนวนที่เลื่อนเข้ามาเป็นดุลยพินิจของนักเทรด — ไม่มีสูตรตายตัว

### ข้อสำคัญ
- **Stop loss ต้องแน่น**: ไม่ควรขาดทุนเกิน 10% ของราคาเข้า (สำหรับหุ้น) หรือจำนวนเงินที่กำหนดไว้ล่วงหน้า (สำหรับฟิวเจอร์ส)

## บทที่ 4: เมื่อไหร่ควร Exit — กฎการออก

ผู้เขียนเน้นย้ำว่า **การออก (exit) สำคัญกว่าการเข้า (entry)** เพราะนี่คือจุดที่ทำเงินจริงๆ

### หลักการสำคัญ

#### 1. ตัดขาดทุนเร็ว (Cut Losses Short)
- ตั้ง stop loss ไว้ล่วงหน้าทันทีที่เข้าเทรด
- **ห้ามคิดว่า "ยังไม่ขาดทุนจนกว่าจะขาย"** — นี่คือความคิดที่อันตรายที่สุด
- **ห้ามคิดว่า "บริษัทดี ทำไมต้องขาย"** — ถ้าราคาลงก็ต้องขาย
- คิดว่า: ถ้าถือตัวที่ขาดทุนอยู่ แล้วมันขวางไม่ให้เราเข้าเทรดตัวที่มีโอกาสทำกำไร → ตัดทิ้งไปแล้วหาตัวใหม่ดีกว่า

#### 2. ปล่อยให้กำไรโต (Let Profits Run)
- **อย่ารีบขายเมื่อมีกำไรเล็กน้อย** — "เงินก้อนใหญ่อยู่ในการเคลื่อนไหวก้อนใหญ่"
- ต้องอดทนรอให้กำไรเล็กๆ กลายเป็นกำไรก้อนใหญ่

#### 3. เทรดน้อยลง (Trade Less)
- ยิ่งเทรดบ่อย ยิ่งเสี่ยง — ช่วงที่อันตรายที่สุดคือตอนเข้าเทรดใหม่
- อดทนรอ Pattern ที่ดีจริงๆ แล้วถือยาว

### กฎ Exit แบบ Step-by-Step

| ระดับกำไร | สิ่งที่ทำ |
|:---------:|:--------|
| เข้าเทรดใหม่ | ตั้ง stop loss แน่น — ห้ามขาดทุนเกิน 10% (หุ้น) หรือ 3% ของทุนทั้งหมด |
| กำไร 25% | ย้าย stop loss ไปที่จุดเข้า (break even) — ถ้าถูกก็แค่เสียค่า commission |
| กำไรเกิน 25% | ปกป้องกำไร 50% (เช่น ถ้ากำไร 30% ให้ตั้ง stop ที่กำไร 15%) |
| กำไร 100% | ปกป้องกำไร 75% |

### จิตวิทยาที่ถูกต้อง
> **กลัวว่าขาดทุนจะใหญ่ขึ้น** → ตัดทิ้งเร็ว
> **หวังว่ากำไรจะใหญ่ขึ้น** → ปล่อยให้วิ่งไกล

**ข้อผิดพลาดของนักเทรดส่วนใหญ่**: ทำตรงข้าม — กลัวกำไรหายเลยรีบขาย (ตัดกำไรเร็ว) และหวังว่าขาดทุนจะกลับมา (ถือขาดทุนนาน)

> **การรีบทำกำไรเร็วเกินไปอันตรายเท่ากับไม่ตัดขาดทุน!**

## บทที่ 5: บทสรุป — "ศัตรูตัวจริงคือตัวเราเอง"

### บทเรียนชีวิตของผู้เขียน
ผู้เริ่มเทรดด้วยการ follow trend + money management ที่ดี → **ทำเงินได้**
จากนั้นก็ออกค้นหา "Holy Grail" (ระบบที่สมบูรณ์แบบ) — เข้าอบรม, ซื้อระบบ $3,000+, ซื้อหนังสือ/หนังสือเสียง/CD นับร้อย → **ไม่มีอะไรใช้ได้จริง 99.5% เป็นของขยะ**

### บทเรียนที่เรียนรู้

1. **ไม่มีใครทำนายอนาคตได้** — ถ้าใครบอกว่าทำนายราคาได้ ให้วิ่งหนี
2. **ความเรียบง่ายคือกุญแจ** — ระบบที่ซับซ้อนเป็นแค่พรางตาให้ยอมจ่ายเงิน
3. **ระบบเทรดมี 3 องค์ประกอบ**:
   - 1) จิตวิทยานักเทรด (Trader psychology)
   - 2) การจัดการเงิน (Money management)
   - 3) ระบบเทรด (Trade system)

### ลำดับความสำคัญที่ถูกต้อง

นักเทรด 95% จะเรียงแบบนี้: ระบบ > เงิน > จิตวิทยา

**แต่ความจริงคือ**:

| ลำดับ | หัวข้อ | เหตุผล |
|:-----:|:------|:------|
| 1 | **จิตวิทยานักเทรด** | ถ้าไม่มี mind set ที่ถูกต้อง ล้มเหลวแน่นอน — อย่าหวังรวยเร็ว อย่าเทรดแบบไม่มีความรู้ (นั่นคือการพนัน) |
| 2 | **การจัดการเงิน** | ผู้เขียนบอกว่าเทรดแค่ด้วย money management อย่างเดียวก็กำไรได้ — สำคัญมากจนต้องใช้เวลาส่วนใหญ่กับส่วนนี้ |
| 3 | **ระบบเทรด** | ระบบที่เรียบง่ายก็เพียงพอ — ไม่ต้องตามหา Holy Grail อีกต่อไป |

### ข้อคิดสุดท้าย
- การเทรดเป็นธุรกิจที่ยอดเยี่ยม **ถ้าลดความเครียดได้**
- อย่าเทรดจากอารมณ์ — อย่าจมกับทุกจังหวะของตลาด
- ใช้ระบบ, money management ที่ดี, และ mind set ที่ถูกต้อง → การเทรดจะสนุกและทำเงินได้จริง
- ขนาดธุรกิจขึ้นอยู่กับเรา — จะเทรดแค่เกษียณก็ได้, จะเทรดเต็มเวลาก็ได้, จะสร้าง empire ก็ได้

## สรุป核心 ของหนังสือทั้งเล่ม

**1-2-3 Pattern** เป็นเครื่องมือระบุจุดกลับตัวที่เรียบง่ายแต่ทรงพลัง:

1. **Pattern**: Bottom → Correction → Retest (ไม่ต่ำกว่า bottom เดิม) → Break correction
2. **เข้าก่อนฝูงชน**: วางออเดอร์ใกล้จุด breakout ไม่ต้องรอ breakout พอดี
3. **จัดการความเสี่ยง**: Stop loss แน่น, ปล่อยกำไรโต, ตัดขาดทุนเร็ว
4. **จิตวิทยาคือสิ่งสำคัญที่สุด** — มากกว่าระบบเทรดและ money management
5. **เรียบง่ายเข้าไว้** — ระบบที่ซับซ้อนไม่ได้ดีกว่าเสมอไป
