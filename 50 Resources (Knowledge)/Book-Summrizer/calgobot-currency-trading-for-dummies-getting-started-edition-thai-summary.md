## Chapter 1: ตลาด Forex คืออะไร?

### ตลาดที่ใหญ่ที่สุดในโลก

ตลาด Forex (Foreign Exchange Market) เป็นตลาดการเงินที่มีการซื้อขายมากที่สุดในโลก โดยมีปริมาณการซื้อขายเฉลี่ยมากกว่า **2 ล้านล้านเหรียญสหรัฐต่อวัน** — มากกว่าตลาดหุ้นทั่วโลกรวมกัน 10-15 เท่า

ตลาด Forex เติบโตขึ้นจากธุรกรรมการค้าระหว่างประเทศ (เช่น บริษัทญี่ปุ่นซื้อชิ้นส่วนจากสวิตเซอร์แลนด์) แต่ปัจจุบัน **กว่า 90% ของปริมาณการซื้อขายมาจากนักเก็งกำไร (speculation)** โดยมีนักเทรดซื้อขายเพื่อทำกำไรจากความผันผวนของราคาในระยะสั้น ธุรกิจและการลงทุนระหว่างประเทศคิดเป็นสัดส่วนน้อยกว่า 10% เท่านั้น

### ลักษณะเด่นของตลาด

- **เปิดทำการ 24 ชั่วโมง 6 วันต่อสัปดาห์** — ตั้งแต่เช้าวันจันทร์ในโซนเอเชีย-แปซิฟิก ไปจนถึงปิดทำการวันศุกร์ที่นิวยอร์ก แม้แต่วันหยุดราชการในประเทศใดก็ตาม ตลาดก็ยังเปิดทำการในประเทศอื่นๆ
- **สภาพคล่องสูงมาก (High Liquidity)** — การซื้อขายมูลค่าครึ่งพันล้านเหรียญสามารถดำเนินการได้ในไม่กี่วินาที โดยแทบไม่กระทบราคา
- **สกุลเงินหลัก (Major Currencies)** คิดเป็น 75% ของปริมาณการซื้อขายทั้งหมด

### สภาพคล่อง (Liquidity)

Liquidity หมายถึงระดับความสนใจของตลาด — ปริมาณการซื้อขายที่มีอยู่ในขณะหนึ่งๆ สำหรับสินทรัพย์ใดสินทรัพย์หนึ่ง ยิ่งสภาพคล่องสูง ราคาก็จะเคลื่อนไหวน้อยลงแม้จะมีปริมาณการซื้อขายมาก ตรงกันข้าม ตลาดที่มีสภาพคล่องต่ำ (thin market) มักจะมีราคาเคลื่อนไหวรุนแรงบนปริมาณการซื้อขายที่น้อยกว่า

### 3 ช่วงเวลาการเทรดหลัก

| ช่วงเวลา | ตลาดหลัก | สัดส่วน Volume | ข้อมูลสำคัญ |
|-----------|----------|---------------|------------|
| **เอเชีย-แปซิฟิก** | Wellington, Sydney, Tokyo, Hong Kong, Singapore | ~21% | ข้อมูลจาก NZ, AU, JP |
| **ยุโรป/ลอนดอน** | London, Frankfurt, Zurich | ~50%+ | ข้อมูลจาก Eurozone, UK, CH |
| **อเมริกาเหนือ** | New York, Toronto | ~22% | ข้อมูลสหรัฐฯ (8:30 ET) |

**ช่วงเวลาที่สภาพคล่องสูงสุด** คือช่วงที่ทับซ้อนกัน — ยุโรปตอนเช้า (ทับกับเอเชียตอนบ่าย) และช่วงข้ามยุโรป-อเมริกา (7:00 ET เป็นต้นไป)

### ความสัมพันธ์กับตลาดการเงินอื่นๆ

- **ทองคำ (Gold)** — โดยทั่วไปมีความสัมพันธ์แบบกลับกันกับ USD (USD อ่อน → ทองขึ้น) แต่ในระยะสั้น แต่ละตลาดมีพลวัตของตัวเอง
- **น้ำมัน (Oil)** — ไม่มีความสัมพันธ์ชัดเจนกับสกุลเงินในระยะสั้น ควรพิจารณาเป็นตัวแปรเงินเฟ้อและปัจจัยจำกัดการเติบโตทางเศรษฐกิจ
- **หุ้น (Stocks)** — หุ้นเป็นหลักทรัพย์ระดับจุลภาค (microeconomic) ตอบสนองต่อผลประกอบการบริษัท ขณะที่สกุลเงินเป็นหลักทรัพย์ระดับมหภาค (macroeconomic) ตอบสนองต่อสถานการณ์เศรษฐกิจและการเมือง ความสัมพันธ์ในระยะยาวแทบเป็นศูนย์
- **พันธบัตร (Bonds)** — มีความเชื่อมโยงกับ Forex มากกว่าเพราะทั้งสองตลาดได้รับผลกระทบจากความคาดหวังอัตราดอกเบี้ย นักเทรดควรติดตาม yield ของพันธบัตรรัฐบาลในประเทศสกุลเงินหลัก

## Chapter 2: กลไกการเทรดค่าเงิน

### การซื้อและขายพร้อมกัน

อุปสรรคทางจิตวิทยาที่ใหญ่ที่สุดสำหรับมือใหม่คือการทำความเข้าใจว่า **การเทรดค่าเงินแต่ละครั้งหมายถึงการซื้อและขายพร้อมกัน** — การซื้อสกุลเงินหนึ่งหมายถึงการขายสกุลเงินอื่นไปในตัว ต่างจากหุ้นที่ซื้อหุ้นแล้วถือไว้ แล้วค่อยขายในภายหลัง

### Currency Pair

สกุลเงินถูกเทรดเป็นคู่ (Currency Pair) โดยมีชื่อเรียกจาก ISO code ของแต่ละสกุลเงิน

**Major Currency Pairs (คู่สกุลเงินหลัก):**

| ISO Pair | ชื่อตลาด | ชื่อเล่น |
|----------|---------|---------|
| EUR/USD | Euro-dollar | — |
| USD/JPY | Dollar-yen | — |
| GBP/USD | Sterling-dollar | Sterling / Cable |
| USD/CHF | Dollar-Swiss | Swissy |
| USD/CAD | Dollar-Canada | Loonie |
| AUD/USD | Australian-dollar | Aussie / Oz |
| NZD/USD | New Zealand-dollar | Kiwi |

**Cross Currency Pairs (คู่สกุลเงินข้าม):** คือคู่ที่ไม่มี USD อยู่ด้วย เช่น EUR/CHF, EUR/GBP, EUR/JPY, GBP/JPY, AUD/JPY ช่วยให้นักเทรดเลือกเทรดได้ตรงจุดกว่า เช่น ถ้ามองว่า JPY อ่อนที่สุด แต่ USD ก็ไม่ได้ดีกว่า อาจเลือก BUY AUD/JPY แทน

### Base Currency และ Counter Currency

- **Base Currency** (สกุลเงินแรก) — เป็นสกุลเงินที่คุณกำลังซื้อหรือขาย และเป็นหน่วยของขนาดเทรด (lot size)
- **Counter Currency** (สกุลเงินที่สอง) — เป็นสกุลเงินที่ใช้อ้างอิงราคา และเป็นหน่วยของกำไร/ขาดทุน

### Going Long vs Going Short

- **Long (ซื้อ)** — ซื้อ currency pair แล้วรอราคาขึ้นเพื่อขายทำกำไร
- **Short (ขาย)** — ขาย currency pair ที่ไม่ได้เป็นเจ้าของ (ใน Forex ไม่ต้องยืมหุ้นเหมือนตลาดหุ้น) แล้วรอราคาลงเพื่อซื้อกลับ
- **Squaring up / Flat** — การปิดสถานะทั้งหมดเพื่อไม่เหลือ position ในตลาด

### Profit and Loss (P&L)

กำไร/ขาดทุนคำนวณจาก **ขนาด position x จำนวน pip ที่เคลื่อนไหว**

**Pip** คือหน่วยการเคลื่อนไหวของราคาน้อยที่สุด:
- คู่สกุลเงินทั่วไป: 4 ตำแหน่งทศนิยม (เช่น EUR/USD 1.285**3**)
- คู่ JPY: 2 ตำแหน่งทศนิยม (เช่น USD/JPY 117.2**3**)

**ตัวอย่างการคำนวณ:**
- BUY 100,000 EUR/USD ที่ 1.2853 → ราคาขึ้นเป็น 1.2873 = +20 pips = **$200 profit**
- SELL 50,000 EUR/USD ที่ 1.2853 → ราคาลงเป็น 1.2792 = +61 pips = **$305 profit**

### Margin และ Leverage

- **Margin** คือเงินประกันที่ต้องวางไว้กับ broker เพื่อเปิด position
- **Leverage** คืออัตราส่วนที่ broker ให้กู้ยืมเพื่อควบคุม position ที่ใหญ่ขึ้น (เช่น 100:1 หมายถึง $1 margin ควบคุม position $100)
- **Margin Balance** ต้องรักษาอัตราส่วน margin ที่ broker กำหนดไว้เสมอ หากต่ำกว่า broker มีสิทธิ์ปิด position ของคุณได้ทันที (Liquidation)
- **Unrealized P&L** = กำไร/ขาดทุนที่ยังไม่ได้ปิด position (mark-to-market)
- **Realized P&L** = กำไร/ขาดทุนที่เกิดขึ้นจริงหลังปิด position

### Rollover และอัตราดอกเบี้ย

เมื่อเปิด position ข้ามวัน (หลัง 5:00 PM ET) broker จะทำ rollover โดยอัตโนมัติ ซึ่งเกี่ยวข้องกับ **อัตราดอกเบี้ยระหว่าง 2 ประเทศ (interest rate differential)**:

- ถ้าคุณ long สกุลที่มีดอกเบี้ยสูง / short สกุลที่มีดอกเบี้ยต่ำ → **ได้เงินrollover**
- ถ้าคุณ short สกุลที่มีดอกเบี้ยสูง / long สกุลที่มีดอกเบี้ยต่ำ → **เสียเงิน rollover**

### Bid/Ask และ Spread

- **Bid** = ราคาที่คุณขาย base currency ได้ (ราคาต่ำกว่า)
- **Ask / Offer** = ราคาที่คุณซื้อ base currency ได้ (ราคาสูงกว่า)
- **Spread** = ความแตกต่างระหว่าง Bid กับ Ask — คือค่าตอบแทนของ broker สำหรับการเป็น market maker
- ยิ่ง liquidity สูง spread ยิ่งแคบ (EUR/USD มักจะแคบสุด 2-3 pips)

## Chapter 3: เลือกสไตล์การเทรดของคุณ

### Technical vs Fundamental Analysis

ไม่มีคำตอบที่ถูกต้องที่สุด — หนังสือแนะนำให้ **ผสมผสานทั้งสองแนวทาง**:

- **Fundamental Analysis** — วิเคราะห์จากข้อมูลเศรษฐกิจ อัตราดอกเบี้ย นโยบายการเงิน กระแสการค้าและการลงทุนระหว่างประเทศ กำหนดทิศทาง大局ของสกุลเงิน
- **Technical Analysis** — วิเคราะห์แผนภูมิ เส้นแนวโน้ม ระดับแนวรับ/แนวต้าน Momentum, Moving Averages ช่วยระบุจุดเข้า/ออกที่แม่นยำ

> Fundamentals บอกทิศทาง大局, Technicals บอกจังหวะเข้า-ออก

### 3 สไตล์การเทรดหลัก

#### 1. Short-term / Scalping (Scalper / Day Trader)

- ถือ position เพียงไม่กี่วินาทีถึง 1 ชั่วโมง
- ทำกำไรจาก pip เล็กๆ แค่ 1-2 pip ต่อเทรด แต่เทรดหลายรอบ
- ต้องมี **ความเร็ว วินัย และสัญชาตญาณ** ในการรับรู้ตลาด
- ต้องเทรดเฉพาะ **คู่ที่ liquidity สูงสุด** (EUR/USD, USD/JPY)
- เทรดเฉพาะ **ช่วง peak liquidity** (2:00 AM - เที่ยง ET)
- หลีกเลี่ยงการถือ position ข้ามข้อมูลเศรษฐกิจสำคัญ

#### 2. Medium-term / Swing Trading (Momentum Trader)

- ถือ position ตั้งแต่ไม่กี่นาทีถึงหลายชั่วโมง (ไม่เกิน 1 วัน)
- ทำกำไรจาก pip 50-100+ ต่อเทรด
- มี 4 แนวทางหลัก:
  - **Trading a View** — ทฤษฎีจากปัจจัยพื้นฐาน (เช่น อัตราดอกเบี้ย)
  - **Trading the Technicals** — จากรูปแบบแผนภูมิและระดับราคา
  - **Trading Events/Data** — เปิด position ก่อนเหตุการณ์สำคัญ
  - **Trading the Flow** — ตามทิศทางของตลาด (trend) หรือข้อมูล Flow
- **ตลาด trend แค่ ~1/3 ของเวลา** ที่เหลืออยู่ในช่วงแกว่ง (range-bound)

#### 3. Long-term / Carry Trade

- ถือ position หลายสัปดาห์ถึงหลายเดือน (แม้แต่หลายปี)
- เหมาะสำหรับ hedge fund และนักลงทุนสถาบัน
- ใช้ leverage น้อย เพื่อทนความผันผวน 5%+ ได้

**Carry Trade Strategy:**
- BUY สกุลเงินที่ yield สูง / SELL สกุลเงินที่ yield ต่ำ
- กำไร 2 ทาง: (1) ได้ interest rate differential (carry) (2) ราคาเคลื่อนไหวไปในทิศทางเดียวกัน
- **ความเสี่ยง**: ราคาอาจพลิกผันรุนแรงกินกำไรทั้งหมด เทรดได้ดีในสภาพแวดล้อมที่ volatility ต่ำ

### Trading Plan — 3 องค์ประกอบหลัก

1. **Position Sizing** — กำหนดขนาด position สำหรับแต่ละเทรด
2. **Entry Level** — จุดเข้าที่แน่นอน (และแผนสำรองหากราคาไม่ถึง)
3. **Stop-Loss & Take-Profit** — จุดออกทั้งฝั่งกำไรและขาดทุน

### วินัยในการเทรด (Trading Discipline)

วินัยคือลักษณะที่สำคัญที่สุดของนักเทรดที่ประสบความสำเร็จ มากกว่าทักษะทางเทคนิคหรือสัญชาตญาณเสียอีก นักเทรดที่มีวินัยสามารถอยู่รอดในตลาดได้ทุกสภาวะ และแม้จะผิดบ่อยกว่าถูก ก็ยังทำกำไรได้

### จัดการอารมณ์ในการเทรด

อารมณ์ไม่สามารถกำจัดได้ ทำได้แค่ **เข้าใจ ยอมรับ และจำกัดผลกระทบ**:

- **โฟกัสที่ pip ไม่ใช่เงิน** — ตลาดไม่รู้ว่าคุณเทรดเท่าไหร่ แต่รู้ว่าราคาอยู่ตรงไหน
- **ไม่ใช่เรื่องถูก/ผิด แต่เป็นเรื่องกำไร/ขาดทุน** — ตลาดไม่สนว่าคุณคิดถูกหรือผิด
- **ยอมรับว่าคุณจะขาดทุนเป็นส่วนหนึ่งของเกม** — นักเทรดที่ประสบความสำเร็จไม่ได้ถูกต้องทุกครั้ง แต่มี plan จัดการความเสี่ยง

## Chapter 4: เริ่มต้นกับ Practice Account

### Practice Account — ขั้นตอนแรก

- เกือบทุก broker ให้บริการ **บัญชีทดลอง (Practice Account) ฟรี** โดยใช้เงิน virtual
- ช่วยให้มือใหม่ได้สัมผัสตลาดจริงโดยไม่ต้องเสี่ยงเงินจริง
- สามารถฝึก: การเปลี่ยนแปลงราคา, ผลกระทบของข้อมูลเศรษฐกิจ, การวิเคราะห์แผนภูมิ, การใช้ leverage, การส่งคำสั่ง
- **สิ่งที่จำลองไม่ได้**: อารมณ์ของการเทรดด้วยเงินจริง — ควรปฏิบัติ practice account เหมือนเงินจริง

### วิธีการเข้าสู่ตลาด (Executing Trades)

1. **Click-and-Deal** — ซื้อขายที่ราคาตลาดปัจจุบันทันที คลิก Buy/Sell บนแพลตฟอร์ม ได้รับการยืนยันภายใน 1-2 วินาที
2. **Using Orders** — ตั้งคำสั่งซื้อขายล่วงหน้าที่จะถูก execute เมื่อราคาถึงระดับที่กำหนด

### ประเภทของ Order (สำคัญมาก)

#### Take-Profit Order
สั่งปิด position เมื่อถึงระดับกำไรที่ต้องการ เช่น BUY GBP/USD ที่ 1.8840 → ตั้ง take-profit ที่ 1.8875

#### Limit Order
สั่งซื้อ/ขายที่ราคาที่ดีกว่าราคาตลาดปัจจุบัน:
- BUY limit: ราคาต่ำกว่าราคาตลาด (ซื้อถูก)
- SELL limit: ราคาสูงกว่าราคาตลาด (ขายแพง)

#### Stop-Loss Order
**สำคัญที่สุดสำหรับความอยู่รอด** — ปิด position เมื่อราคาเคลื่อนไหวสวนทาง:
- Long position → stop-loss เป็น SELL ที่ราคาต่ำกว่า
- Short position → stop-loss เป็น BUY ที่ราคาสูงกว่า

#### Trailing Stop-Loss
stop-loss ที่ **เลื่อนตามราคาในทิศทางเดียวกัน** เท่านั้น:
- BUY EUR/CHF ที่ 1.5750, trailing stop 30 pips → stop เริ่มที่ 1.5720
- ถ้าราคาขึ้นเป็น 1.5775 → stop เลื่อนขึ้นเป็น 1.5745
- ถ้าราคาลง 30 pips จากจุดสูงสุด → trigger stop-loss

#### OCO Order (One-Cancels-the-Other)
**ประกันภัยที่สมบูรณ์** — วาง Stop-Loss + Take-Profit คู่กัน ถ้า触发 อันหนึ่ง อีกอันจะถูกยกเลิกอัตโนมัติ ป้องกันไม่ให้ position เปิดค้างโดยไม่มี protection

#### Contingent Order (If/Then)
组合คำสั่งหลายประเภทเข้าด้วยกันเป็นกลยุทธ์完整 เช่น "ถ้า BUY ถึง A → ตั้ง stop-loss ที่ B และ take-profit ที่ C"

### การจัดการ Trade หลังเปิด Position

- **ติดตามข่าวสารและเหตุการณ์** ที่กระทบต่อ position ของคุณ
- **ปรับ plan เฉพาะในทิศทางเดียว** — เอา profit หรือย้าย stop-loss เข้าหาจุดทำกำไร ไม่ขยับ stop-loss ออกห่าง
- **ติดตามตลาดอื่นๆ** (หุ้น, พันธบัตร, น้ำมัน) เพื่อยืนยันหรือคัดค้านทิศทางของคุณ

### การประเมินผลการเทรด

หลังปิดทุกเทรด ควรถามคำถามเหล่านี้:

1. **ระบุโอกาสเทรดได้อย่างไร?** — ใช้ technical, fundamental หรือผสม? ช่วยระบุจุดแข็ง/อ่อน
2. **Plan ทำงานได้ดีแค่ไหน?** — Position size, entry level, stop-loss/take-profit สมเหตุสมผลหรือไม่?
3. **จัดการเทรดหลังเปิดได้ดีแค่ไหน?** — ติดตามตลาดได้ดีไหม? ปรับ plan ไหม? ทำตามวินัยไหม?

> **สิ่งที่สำคัญที่สุด**: ซื่อสัตย์กับตัวเอง ไม่มีคำตอบที่ถูก/ผิด แค่ต้องเรียนรู้จากทุกเทรดเพื่อปรับปรุงอย่างต่อเนื่อง

## สรุป Key Takeaways

| หัวข้อ | ประเด็นสำคัญ |
|--------|-------------|
| ตลาด Forex | ใหญ่ที่สุดในโลก 2 ล้านล้าน$/วัน, liquidity สูง, เปิด 24 ชม. |
| Currency Pair | สกุลเงินเทรดเป็นคู่, Base/Counter currency, Major vs Cross |
| Pip | หน่วยราคาเล็กสุด คำนวณ P&L จาก size × pips |
| Margin/Leverage | วางเงินประกันเล็กๆ ควบคุม position ใหญ่ |
| Bid/Ask/Spread | Spread คือค่าตอบแทน broker, liquidity สูง spread แคบ |
| 3 Trading Styles | Scalping (สั้น), Swing (กลาง), Carry Trade (ยาว) |
| Trading Plan | Position size + Entry + Stop-Loss/Take-Profit |
| Order Types | Stop-Loss สำคัญสุด, Trailing Stop ปล่อยกำไรวิ่ง, OCO ประกันครบ |
| วินัย | สำคัญกว่าทักษะเทคนิค ทำตาม plan ทุกครั้ง |
| อารมณ์ | กำจัดไม่ได้ แค่จำกัดผลกระทบ โฟกัสที่ pip ไม่ใช่เงิน |
